@page embedding Embedding MantisBase

MantisBase is designed as a lightweight C++ library that can be embedded directly into your desktop, mobile, or embedded application. This guide explains how to integrate MantisBase into your project, configure it, and expose APIs through your own app.

## Dependencies
On Linux, MantisBase depends on ZSTD and libpq for PostgreSQL linkage, both of which a can be installed by running the commands below;

```shell
sudo apt-get install -y libzstd-dev
sudo apt install libpq-dev
```

## ğŸ“¦ Integration Options

### **As a Static or Shared Library**
Link the compiled `libmantisbase.a` or `libmantisbase.so` into your CMake project:

```cmake
add_subdirectory(mantisbase)
target_link_libraries(your_app PRIVATE mantisbase)
```
