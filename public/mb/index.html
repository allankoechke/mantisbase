MantisBase Admin

Legacy placeholder only. The server serves the compiled admin app from public/mb-dist/ at /mb/ — run cd admin && npm install && npm run build.

Use HTTP Basic auth with an admin account on /api/v1/sys/* (schemas, configs, logs), /api/v1/admins/* (admin user list / create / delete), and similar protected routes. Public and user-JWT routes stay under /api/v1/entities/*, /api/v1/auth/*, etc.
